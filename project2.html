Project2 : Change my rabbit character

Interaction using processing


작품 소개 :
토끼 캐릭터가 왼쪽에 그려져 있으며, 화면에 나와있듯이 키보드의 'a','s','d','f'를 각각 누르면 캐릭터의 모양이 바뀐다.
또한, 랜덤의 좌표값에 따라 당근 이미지가 나타난다. - 주요 함수 : keyType()
'a' : 캐릭터 입 모양이 바뀜
's' : 캐릭터 눈 모양이 바뀜
'd' : 캐릭터 귀 색깔이 바뀜
'f' : 오른쪽 귀 옆에 꽃이 나타남

아래의 그림을 클릭하면 작품이 나타납니다.
